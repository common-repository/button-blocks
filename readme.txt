== Description ==

The Button Blocks plugin brings a simple yet powerful button block to your WordPress editor, allowing you to easily create call-to-action buttons with full control over alignment, text, URLs, and additional options. Whether you're a beginner or an experienced developer, this block gives you the flexibility to customize buttons that match your site's design and functionality.

Key Features:

Customizable Alignment: Adjust the button's position to left, right, or center to align perfectly with your content.
Editable Text: Change the button's label to suit your call-to-action needs, whether it's "Learn More," "Buy Now," or anything else.
URL Support: Add custom URLs to make your button clickable and direct users to specific pages, forms, or external websites.
Target Attribute: Choose whether the link opens in the same window or in a new tab for more control over navigation.
Styling Options: Customize the appearance of your button to match your theme. Change colors, sizes, and other styles for a unique look.
Responsive Design: Your buttons will automatically adapt to different screen sizes, ensuring an optimal display on mobile, tablet, and desktop devices.
User-Friendly Interface: Designed for the WordPress block editor (Gutenberg), the Button Blocks plugin integrates smoothly into the editor experience, providing intuitive settings without the need for additional coding.

== Installation ==

There is no prerequisites for installing **Button Blocks**. You can follow the steps below to complete the installation.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Button Blocks`.
3.  Install and activate the plugin, that's it!
